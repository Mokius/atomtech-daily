# React Navigation 8.0 Alpha Ships with Native Bottom Tabs Default

> **💻 Software Engineering** · 🟢 Notable · 2026-04-27

---

## 📋 Summary
React Navigation version 8.0 has entered alpha, bringing major architectural changes to the most widely used routing library for React Native and web applications. Key changes include native bottom tabs as the default tab navigator, significantly enhanced TypeScript inference for type-safe navigation, and improved performance for deep-linking scenarios. The alpha is available now for testing on both React Native and web targets.

## 💡 Why It Matters
React Navigation is used in the vast majority of production React Native apps. Moving to native bottom tabs by default aligns more closely with Apple and Google platform guidelines, resulting in better performance and native-feel UX without manual configuration.

## 🔍 What Is This?
React Navigation manages the screens in a mobile app — handling how users move between pages, go back, open modals, and navigate tab bars. Version 8.0 makes navigation feel more naturally native on both iOS and Android.

## 🔭 Looking Ahead
The stable 8.0 release is expected in Q2 2026. App developers should begin testing now to identify migration requirements before the stable release lands.
