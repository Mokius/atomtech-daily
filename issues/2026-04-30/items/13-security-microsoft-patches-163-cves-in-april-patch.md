# 🔐 Microsoft Patches 163 CVEs in April Patch Tuesday Including Active SharePoint Zero-Day

> **🔐 Security** · 🔴 High · 2026-04-30

---

## 📋 Summary
Microsoft's April 2026 Patch Tuesday addressed 163 CVEs, including CVE-2026-32201 — a SharePoint Server spoofing zero-day with a CVSS score of 6.5 that has been actively exploited in the wild. The vulnerability allows unauthenticated remote attackers to perform spoofing via improper input validation, requiring no user interaction. CISA has already added it to its Known Exploited Vulnerabilities (KEV) list, mandating federal agencies to patch by April 28.

## 💡 Why It Matters
CVE-2026-32201 is particularly dangerous because it requires no authentication and no user interaction, meaning attackers can exploit it without any victim involvement. SharePoint is widely deployed across enterprise and government environments, making the blast radius of unpatched systems significant. The CISA KEV listing means federal agencies face compliance deadlines.

## 🔍 What Is This?
Patch Tuesday is Microsoft's monthly security update release. A zero-day vulnerability is a security flaw that attackers are actively exploiting before a patch is available — or immediately after, before organizations can apply the fix. SharePoint is Microsoft's widely-used enterprise collaboration and document management platform.

## 🔭 Looking Ahead
Organizations running SharePoint Server should treat this as an emergency patch priority. The combination of no authentication required, no user interaction, and active exploitation makes this one of the higher-urgency CVEs in recent months, regardless of its moderate CVSS score.
