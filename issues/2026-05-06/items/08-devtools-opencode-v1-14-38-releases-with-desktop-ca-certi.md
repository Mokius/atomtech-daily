# 💻 opencode v1.14.38 Releases with Desktop CA Certificate Support and Improved Security

> **💻 Engineering & Dev Tools** · 🟢 Notable · 2026-05-06

---

## 📋 Summary
The open-source AI coding tool opencode released version v1.14.38 on May 5, 2026. Key updates include embedded UI requests that now work with arbitrary connect-src origins under the default Content Security Policy, and desktop system CA certificate support — allowing the tool to respect enterprise certificate authorities in corporate environments. The release improves opencode's viability in enterprise and air-gapped deployment scenarios.

## 💡 Why It Matters
Enterprise adoption of open-source AI coding tools has been blocked by security policy requirements, particularly around certificate management and Content Security Policy restrictions. The CA certificate support in v1.14.38 directly addresses a common blocker for IT departments evaluating opencode as a GitHub Copilot alternative. This signals the project is maturing beyond personal use toward production enterprise deployment.

## 🔍 What Is This?
opencode is a free, open-source AI coding assistant — similar to GitHub Copilot or Cursor — that developers can run locally. This update makes it work properly in corporate environments where IT teams have custom security certificates and strict web content policies.

## 🔭 Looking Ahead
With enterprise compatibility improving, opencode is becoming a realistic option for organizations that cannot use cloud-hosted coding assistants due to data privacy requirements. Expect more enterprise-focused features in upcoming releases.
