# 💻 MySQL 9.7.0 General Availability Launches First New LTS Release Series Since 8.4

> **💻 Dev Tools** · 🟡 Medium · 2026-05-11

---

## 📋 Summary
Oracle announced the General Availability of MySQL 9.7.0, marking the start of a new 9.7 Long-Term Support (LTS) release series and the first major LTS milestone since MySQL 8.4. The 9.7 series brings performance improvements, expanded JSON capabilities, enhanced replication features, and improved support for modern application architectures. As a LTS release, MySQL 9.7.0 will receive extended support and security patches, making it suitable for production workloads requiring long-term stability.

## 💡 Why It Matters
MySQL powers a significant portion of the world's web infrastructure — billions of applications depend on it. A new LTS release series gives enterprises a stable, long-supported version to migrate to from older releases. The enhancements to JSON handling and replication are particularly relevant as developers increasingly use relational databases for semi-structured data and distributed architectures.

## 🔍 What Is This?
MySQL is one of the most popular open-source databases in the world, used by companies like Facebook, Twitter, and countless web applications. A Long-Term Support (LTS) release means this version will receive security updates and bug fixes for an extended period — typically 5+ years — making it safe for businesses to build on.

## 🔭 Looking Ahead
MySQL 9.7 LTS will likely become the standard production version for enterprises over the next 12-18 months as teams migrate from 8.x releases. The enhanced JSON capabilities signal Oracle's commitment to keeping MySQL competitive with document-oriented databases for hybrid data models.
