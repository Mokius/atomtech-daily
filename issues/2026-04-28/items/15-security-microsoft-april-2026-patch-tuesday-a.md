# 🔐 Microsoft April 2026 Patch Tuesday Addresses 163 CVEs Including SharePoint Zero-Day

> **🔐 Security** · 🔴 High · 2026-04-28

---

## 📋 Summary
Microsoft's April 2026 Patch Tuesday addressed 163 CVEs including eight Critical-severity vulnerabilities and one actively exploited zero-day: CVE-2026-32201, an improper input validation flaw in Microsoft SharePoint that allows an unauthenticated attacker to perform spoofing attacks over a network. The patch also addressed a previously publicly disclosed zero-day that had been known to attackers before today's fix.

## 💡 Why It Matters
163 CVEs in a single patch cycle is among the largest Patch Tuesday releases on record, reflecting both the expanding attack surface of Microsoft's product portfolio and the intensity of modern vulnerability research. The SharePoint zero-day (CVE-2026-32201) is particularly significant for enterprises with on-premises SharePoint deployments and hybrid Microsoft 365 environments, where network-accessible spoofing can enable identity-based attacks.

## 🔍 What Is This?
Patch Tuesday is Microsoft's monthly security update release, typically on the second Tuesday of each month. A CVE (Common Vulnerabilities and Exposures) is a catalogued security flaw. 163 patches in one month is extraordinarily high. The SharePoint zero-day means attackers were already actively exploiting this vulnerability before today's fix — organizations with SharePoint deployments should treat this as a priority patch.

## 🔭 Looking Ahead
With 163 CVEs addressed and three additional Defender zero-days still in various patch states, April 2026 is an unusually demanding month for Windows security operations teams. Organizations should prioritize CVE-2026-32201 for immediate deployment while auditing their full patch status across the broader Patch Tuesday release.
